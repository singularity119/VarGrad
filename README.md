# Towards Calibrated Gradient-based Multi-Task Learning

This repository provides code for the paper **"Towards Calibrated Gradient-based Multi-Task Learning"** (anonymous submission).
The current release includes code for experiments on the **Cityscapes** dataset.

## Dataset Setup

We follow the preprocessing protocol from the [MTAN paper](https://github.com/lorenmt/mtan).

### Cityscapes

The preprocessed Cityscapes dataset used in our experiments can be downloaded from the following link:

🔗 [Download Cityscapes (preprocessed)](https://www.dropbox.com/scl/fo/x2i67p14fxy3d3178i8ln/AGZHvvk82ayNbcBHp-N0JXg?rlkey=ebhrpay2cgqnr0ew5vmti2diy&e=1&dl=0)

After downloading, place the dataset under the following directory:

```
experiments/cityscapes/dataset/
```

Make sure the structure matches what is expected by the code.
